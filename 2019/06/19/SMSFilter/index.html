Message Filtter Extension 短信过滤
发布于 2019-06-19   更新于 2019-06-25 热度 0 喜欢 0

手机最近老是收到各种垃圾短信，于是想要自己写一个app来过滤垃圾短信，找到了Message Filtter Extension

1.首先创建一个空的工程，然后创建好了之后添加一个target。

2.添加好了之后选择设置，项目工程和target都要开启AppGroup,点击左下角的加号添加一个AppGroup的标示

注意：项目工程和自己添加的target必须要一致，这个标示是用来共享数据的

3.工程这边做一个简单的列表用来添加屏蔽关键词，然后关键词的本地化用之前添加的group.com.message.fliter标示用userdefault来存储

4.在Message Filtter Extension实现关键词过滤,而这个关键词的数据是从项目工程里拿的。这就实现了短信的过滤，不过遗憾的是没有办法可以删除垃圾短信。

这边添加了一个简书的关键词.
